🤖 Telegram Bot

Add @GoodExplorerBot on Telegram to compare grocery prices right from your chat.

📷 QR Code Placeholder — scan to open bot
Setup: Create a bot via @BotFather, set TELEGRAM_BOT_TOKEN, then run good-explorer telegram-bot for polling or configure the webhook URL above.

💬 Add to Slack

Use the /compare slash command in Slack to get instant price comparisons.

Add to Slack
Setup: Create an Slack App, add a slash command pointing to /api/slack/command, and optionally create an incoming webhook for proactive posting.

📱 WhatsApp

Message us on WhatsApp to compare prices on the go.

Message on WhatsApp
Setup: Use Twilio WhatsApp Sandbox or WhatsApp Business API. Point the webhook to /api/whatsapp/webhook. Set TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_WHATSAPP_FROM for outbound messaging.
